# Slug Generator API (NestJS / Node.js)

A production-quality REST API that converts text into a URL-friendly slug.

Example:
"Hello World!" → "hello-world"

---

## 🚀 Features

- Convert text to URL-safe slugs
- Clean and consistent slug rules
- RESTful API design
- Unit-tested core logic
- Built with NestJS (or Node.js + Express)
